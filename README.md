# 최성빈

문제를 정의하고 끝까지 해결해내는 개발자입니다.  
사용자의 불편함을 해소하기 위해 기술의 경계를 두지 않습니다.  
웹 서비스 구현부터 AI 모델 활용, 인프라 구축까지 필요한 기술을 주도적으로 학습하고 적용합니다.  
복잡한 요구사항을 명확한 로직으로 구현하며, 안정적인 서비스 운영을 최우선 가치로 둡니다.
